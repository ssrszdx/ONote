# 正则表达式

* 括号

  1、小括号（）：匹配 小括号内的字符串，可以是一个，也可以是多个；

  　　　　　　　　常跟“|”(或)符号配合使用。

  　　　　　　　　例如  String text="123456789";

  　　　　　　　　　　text.matches("(0-9){9}")  ----false //匹配小括号里的字符串'0-9' 出现9次

  　　　　　　　　　　text.matches("[0-9]{9}")  ----true // 匹配0到9的任意字符，正好出现9次

  　　　　　　　　　　"0-9".macthes("(0-9)")  true

  2、中括号：匹配字符组内的字符

  　　　　　　例：[0-9] 代表0-9内的任意字符

  　　　　　　　　"0".matches([0-9]) true

  3、大括号：匹配该字符出现的次数

  　　　　　　[0]{n}------> 0正好出现n次

  　　　　　　[0]{n,m}----->0最少出现n次，最多出现m次
